# **Description des scripts du Projet bloc 1**

- Script extraction_donnees.py : Récupération des données sur le site paru vendu
- Script insertion_des_donnees_dans_la_bdd_immobilier.py : Insertion des données dans la base de données immobilier
- Script lecture_des_donnees_dans_la_bdd_immobilier.py : Lecture des données dans la base de données immobilier 
- Script authentification_api.py : Authentification à l'api, écriture de la documentation de l'api
